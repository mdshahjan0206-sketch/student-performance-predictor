# Student Performance Predictor

A production-grade ML-powered platform for predicting student academic performance from uploaded datasets.

## Features
- Upload any student dataset with target column selection
- Automatic feature detection and schema validation
- Train Random Forest, Decision Tree, and Logistic Regression models
- Interactive EDA and model metrics dashboard
- Export predictions and history
- Responsive, dark/light mode UI

## Setup
```bash
pip install -r requirements.txt
streamlit run app.py
```
